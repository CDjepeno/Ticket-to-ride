# Getting Started
```bash
$ git clone https://github.com/CDjepeno/ticket_to_ride.git
$ cd ticket_to_ride
$ npm install
$ Configure file ormconfig.json for database mysql
$ npm run test
$ npm run typeorm:create
$ npm run typeorm:run 
$ npm run dev
$ http://localhost:3000/swagger/ => To see all API
```
